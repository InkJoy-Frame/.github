# InkJoy Open Apps

**Build creative applications for InkJoy Frame — easily, even with AI.**

This organization hosts open-source applications, examples, and experiments built on top of the InkJoy Open API.

---

## 🚀 Why InkJoy Open API

The InkJoy Open API is designed to be:

- ⚡ Simple and easy to use  
- 🤖 Friendly for AI-assisted development  
- 🎨 Flexible for creative applications  

In fact, you can use AI tools to quickly build your own app and bring your ideas to life on an InkJoy Frame.

👉 Start here:  
[https://app.inkjoyframe.com/statics/developer/](https://openapi.inkjoyframe.com/)

---

## 🧩 What You Can Do

With the current Open API, you can:

- 📡 Query the status of your InkJoy Frames  
- 🖼 Push any image to your frame  
- ☁️ Let the cloud handle rendering via ISFR  
- 🎯 Deliver the processed image to a specific device  

This means you can focus on **ideas and experiences**, while InkJoy handles the rendering complexity.
